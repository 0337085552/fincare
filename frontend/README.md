# Frontend - Quản lí chi tiêu

Giao diện frontend thuần HTML, CSS, JavaScript cho website quản lí chi tiêu cá nhân. Dự án được để sẵn cấu trúc giao diện, form, bảng, modal và các khu vực nội dung để bạn nối backend sau.

## Chạy dự án

```bash
cd frontend
npm install
npm run dev
```

Mở trình duyệt tại: http://localhost:3000

## Ghi chú tích hợp backend

- File chính: `src/main.js`
- Style chính: `src/style.css`
- Các hàm mẫu để nối API nằm ở cuối `src/main.js`: `apiGet`, `apiPost`, `apiPut`, `apiDelete`
- Dữ liệu hiện tại chỉ là placeholder hiển thị giao diện, chưa lưu thật.
